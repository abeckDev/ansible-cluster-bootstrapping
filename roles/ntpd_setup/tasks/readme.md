# task Ordner  

Im Task Ordner befinden sich die Hauptaufgaben, welche das Ansible im Rahmen der Rolle ausführen soll.

## Beispiele

Abhängig je nach Funktion des Playbooks

## Verwendung

In .yml Datein werden die Aufgaben beschrieben
